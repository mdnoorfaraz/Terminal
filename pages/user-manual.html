⚙️ Terminal Web App

📘 User Manual

Welcome to your Terminal Web App. Here's a guide to all available commands and features.

🧾 General Commands

  • help – List all available commands
  • clear or cls – Clear the terminal screen
  • exit – Exit or redirect from terminal page
  • whoami – Show current user info if logged in

🔐 Authentication

  • login – Start login process (Google or Email)
  • logout – Log out current user

🧭 Web Launcher

  • map – Opens Google Maps
  • insta – Opens Instagram
  • movies – Opens Movies Page

📁 File System

  • upload – Upload files to Firebase Storage (coming soon)
  • download – List and download uploaded files

💬 Messaging & Notes

  • note add [text] – Add a personal note
  • note show – Display saved notes
  • msg send [username] [message] – Send message to user
  • msg inbox – Check your inbox

🎨 Themes & Personalization

  • dark – Switch to dark mode
  • light – Switch to light mode
  • theme hacker, theme classic – Set specific themes (optional)
